## Pré-requisitos

Para executar o projeto, é necessário ter o **FPGRARS** instalado em sua máquina.
Uma versão funcional já está incluída neste repositório.
Caso prefira baixar a versão oficial, ela está disponível no [repositório original](https://github.com/LeoRiether/FPGRARS).

---

## Como jogar

1. **Clone o repositório na sua máquina:**

```bash
git clone https://github.com/Paulohpr/TheLegendOfCLASH_ProjetoISC25.2
```

Ou clique em **Code** e selecione **Download ZIP**.

2. **Abra o projeto no FPGRARS:**

   * Localize o arquivo `jogo.s`
   * Abra ou mova o arquivo diretamente para o **FPGRARS**
